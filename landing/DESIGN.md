# DESIGN.md — «Единое окно» (одно-окно.рф)

Токены и дизайн-направление лендинга. Читать ПЕРВЫМ перед любой версткой.
Источник направления: `7.html` Андрея + `_research/FOUNDATION.md` §5.

## Направление
**Индустриально-инженерное «деловое техно»** для центра легализации недвижимости.
Метафора-объект: полевой журнал геодезиста / кадастровый чертёж / технический регламент.
Тон: точный, авторитетный, без воды. Слоган-якорь: «Мы не консультируем. Мы легализуем.»
Сигнатура (то, что запомнят): **blueprint-сетка + данные как декор** (кадастровые номера, координаты, мономаркеры как часть графики).

## Тема
Светлая база (платиновый фон) с тёмно-синими и «приборными» тёмными блоками. Не дефолтный лайт ради безопасности — выбран под дневной деловой просмотр + контраст инженерных блоков.

## Шрифты (все с кириллицей, Google Fonts)
- **Unbounded** (600–900) — дисплей/крупные заголовки. Геометрический, архитектурный. Использовать сдержанно.
- **Golos Text** (400–800) — основной текст и UI. Институциональный, доверие.
- **JetBrains Mono** (400–700) — кадастровые номера, координаты, технические метки, шаги процесса.
Запрещены: Inter, Montserrat, Lato, Fraunces (нет кириллицы), Roboto, system-ui.

## Цвет (oklch, перцептивно ровные; на светлом — низкая chroma)
- `--bg`            oklch(0.968 0.006 247)   /* платиновый фон #F1F5F9 */
- `--surface`       oklch(0.992 0.002 247)   /* «белые» карточки, тинтованы */
- `--navy`          oklch(0.255 0.055 252)   /* deep navy #001A33, тёмные блоки/текст */
- `--navy-deep`     oklch(0.185 0.045 252)   /* подвал #000F21 */
- `--cobalt`        oklch(0.455 0.155 258)   /* #0047AB линии/графика */
- `--tech-blue`     oklch(0.625 0.185 252)   /* #007FFF интерактив/hover */
- `--accent`        oklch(0.82 0.165 84)     /* industrial yellow #FFB800 — CTA, 10% веса */
- `--ink`           oklch(0.255 0.055 252)   /* основной текст = navy */
- `--ink-soft`      oklch(0.255 0.04 252 / 0.72)
- `--line`          oklch(0.455 0.155 258 / 0.16) /* blueprint-линии */
Правило 60/30/10: фон/поверхность 60, текст/линии 30, жёлтый акцент 10 (редко = сильно).
Без чистого #000/#fff. Без cyan-on-dark, без purple→blue градиентов, без gradient-text.

## Тип-шкала (fluid clamp, ratio ~1.3)
- display  clamp(2.6rem, 1.6rem + 4.5vw, 5.25rem)  / line-height 1.02 / Unbounded 800
- h2       clamp(1.9rem, 1.3rem + 2.4vw, 3rem)      / Unbounded 700
- h3       clamp(1.15rem, 1rem + 0.7vw, 1.45rem)    / Golos 700
- body     clamp(1rem, 0.96rem + 0.2vw, 1.1rem)     / Golos 400, line-height 1.6, max 68ch
- label    0.74rem / JetBrains Mono 500 / uppercase / letter-spacing 0.12em

## Пространство (4pt, семантика)
4 8 12 16 24 32 48 64 96 128 → `--s-1..--s-10`. Ритм варьируем: секции дышат (clamp паддинги),
группы плотные. `gap` вместо margin. НЕ одинаковый padding везде.

## Моушн
Один срежиссированный вход (stagger reveal hero), hover-состояния. ease-out-quint.
Только transform/opacity. Без bounce. `prefers-reduced-motion` — выключить.

## Анти-slop (для ЭТОГО направления)
- Hero НЕ центрированный + 3 карточки. Асимметрия: крупный левый заголовок + правый «приборный» блок.
- Сетка «болей» НЕ одинаковые карточки — bento/разноразмерная.
- НЕТ border-left цветных полосок, НЕТ gradient-text, НЕТ glass везде, НЕТ иконок-эмодзи.
- Blueprint-сетка и мономаркеры — осмысленный бренд-приём, не декор ради декора.
- Скруглания минимальные (2–4px), прямоугольная инженерная геометрия.
